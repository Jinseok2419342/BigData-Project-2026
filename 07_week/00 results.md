### llm_json.py 원본 코드 
```
import ollama
import json

reviews = [
    "배송 빠르고 제품 품질도 좋아요! 재구매 의사 있습니다.",
    "불량품이 왔는데 교환도 안 해주네요. 최악입니다.",
    "가격은 저렴한데 품질은 보통이에요."
]

results = []

for review in reviews:
    response = ollama.chat(
        model = "gemma3:4b",
        messages=[
            {
                "role": "system",
                "content": """당신은 리뷰 분석 전문가입니다.\n주어진 리뷰를 분석하여 반드시 아래 JSON 형식으로만 응답\n다른 텍스트는 포함하지 마세요.\n{"sentiment": "긍정/부정/중립", "confidence": 0.0~1.0, "keywords": ["키워드1", "키워드2"]}"""
            },
            {
                "role": "user",
                "content": review
            }
        ]
    )

    raw = response["message"]["content"]

    # JSON 파싱 시도
    try:
        # LLM이 ```json ... ``` 으로 감싸서 응답할 수 있음
        clean = raw.strip()
        if "```json" in clean:
            clean = clean.split("```json")[1].split("```")[0].strip()
        elif "```" in clean:
            clean = clean.split("```")[1].split("```")[0].strip()

        data = json.loads(clean)
        results.append(data)
        print(f"리뷰: {review[:25]}...")
        print(f"   감성: {data['sentiment']}, 확신도: {data['confidence']}")
        print(f"   키워드: {data['keywords']}")
    except json.JSONDecodeError:
        print(f"JSON 파싱 실패: {raw[:100]}")

    print("-" * 50)

print(f"\n총 {len(results)}개 리뷰 분석 완료")
```

### llm_sentiment.py 원본 코드

import ollama

reviews = [
    "재미있었어요! 연기도 훌륭!",
    "시간 낭비. 스토리 엉망",
    "그냥 그랬어요.",
    "OST 좋지만 결말이 아쉬워요"
]

for review in reviews:
    response = ollama.chat(
        model = "gemma3:4b",
        messages = [
            {
                "role" : "system",
                "content" : "감정 분석 전문가. 긍정/부정/중립"
            },
            {
                "role" : "user",
                "content" : review
            }
        ]
    )

    print(f"리뷰: {review[:30]}...")
    print(f"분석: {response['message']['content']}")
    print("-" * 60)
```
### 원본 답변 
```
리뷰: 재미있었어요! 연기도 훌륭!...
분석: 정말 긍정적인 반응이네요! 

**긍정** 감정입니다. 😊 

배우의 연기에 대한 만족감을 표현해주셔서 감사합니다!
------------------------------------------------------------
리뷰: 시간 낭비. 스토리 엉망...
분석: 네, 분석 결과는 다음과 같습니다.

*   **감정:** 부정 (Negative)
*   **설명:** "시간 낭비"와 "스토리 엉망"이라는 표현은 명확하게 불만족스러운 감정을 나타냅니다. 스토리의 품질에 대한 비판적인 의견을 담고 있어 부정적인 감정이 강하게 나타납니다.
------------------------------------------------------------
리뷰: 그냥 그랬어요....
분석: 이 문장은 **중립**적인 감정을 나타냅니다. 

특정 긍정적 또는 부정적 감정이 느껴지지 않고, 단순히 "그랬다"는 사실만을 전달하고 있기 때문입니다. 

좀 더 구체적인 감정을 표현해주시면 더 정확한 분석을 도와드릴 수 있습니다. 예를 들어, "그냥 그랬어요, 조금 실망했어요" 와 같이 덧붙여 주시면 됩니다.
------------------------------------------------------------
리뷰: OST 좋지만 결말이 아쉬워요...
분석: **분석:** 부정적인 감정이 강하게 드러납니다. 

*   **주요 감정:** 아쉬움, 실망
*   **세부 분석:** OST는 긍정적인 요소로 긍정적인 평가를 받았지만, 결말에 대한 실망감이 전반적인 평가를 부정적으로 만들고 있습니다.

**전반적인 평가:** 6/10 (음악 자체는 좋았지만 결말 때문에 만족도가 낮음)
------------------------------------------------------------
```

import ollama

# 리뷰 → 카테고리 분류를 Few-shot으로
fewshot_prompt = """다음은 상품 리뷰를 카테고리로 분류하는 예시입니다.

리뷰: "배송이 너무 빨라서 좋았어요"
카테고리: 배송

리뷰: "가격 대비 품질이 좋습니다"
카테고리: 품질

리뷰: "고객센터 응대가 친절했습니다"
카테고리: 서비스

리뷰: "포장이 꼼꼼하게 되어 있었어요"
카테고리: 배송

이제 다음 리뷰를 분류해주세요. 카테고리만 답하세요.

리뷰: "사용법을 물어봤는데 자세히 알려줘서 감사합니다"
카테고리:"""

response = ollama.generate(model = "gemma3:4b", prompt = fewshot_prompt)
print("분류 결과:", response["response"].strip())

fewshot_prompt = """당신은 카페 리뷰를 분석하는 전문가입니다.
아래 예시를 참고하여 리뷰를 적절한 카테고리로 분류해주세요.

리뷰: "라떼가 진하고 맛있었어요"
카테고리: 맛

리뷰: "카페가 너무 시끄러워서 집중이 안 됐어요"
카테고리: 분위기

리뷰: "직원분이 친절하고 빠르게 서빙해 주셨어요"
카테고리: 서비스

리뷰: "케이크 양이 너무 적었어요"
카테고리: 양

리뷰: "커피가 식어서 나왔어요"
카테고리: 온도

이제 다음 리뷰를 보고, **카테고리 하나만** 정확하게 답변해주세요.
답변은 "카테고리: XXX" 형식으로만 작성하고, 다른 설명은 하지 마세요.

리뷰: "직원이 메뉴 추천을 자세히 해줘서 고르기 쉬웠어요"
카테고리:"""

response = ollama.generate(model = "gemma3:12b", prompt = fewshot_prompt)
print("분류 결과:", response["response"].strip())
```
